Gemini CLI Reference

Optimized for Desktop & Mobile Terminal Masters

1. Interactive Slash Commands

Use these inside your live CLI session.

Session Management

Command Purpose
/helpShow all available commands.
/clearReset the visual terminal and context.
/quitExit the AI agent session.
/chat save <tag>Save history under a tag.
/chat resume <tag>Reload a saved session.
/chat listList all stored session tags.

Tools & Memory

Command Purpose
/toolsList all active function tools.
/mcpManage external data servers.
/memory add <text>Save custom prompt rules.
/memory showView active guardrails & style.
/modelSwitch between AI models.

2. Startup Flags

Terminal arguments for launching the tool.

Flag Effect
-p "query"One-shot prompt and exit.
-i "query"Start with prompt and stay interactive.
-m "<name>"Override default model choice.
-r "latest"Auto-resume the last session.
--yoloEnable automatic tool execution.

3. Special Triggers

Input prefix notation for direct actions.

Symbol Action
@<path>Inject file/folder into context.
!<cmd>Execute local shell command.
